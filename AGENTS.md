# AI Assistant Project Rules

1. **Strictly adhere to user requests**: Only implement the changes or features explicitly requested by the user. Do not add unrequested features or "improve" code outside the scope of the prompt.
2. **Preserve existing functionality**: Do not break, refactor, or delete existing code unless explicitly instructed to do so. Assume the current code state is intended by the user.
3. **No sweeping UI changes**: Do not alter styling, layouts, animations, or themes unless the user specifically asks for those design changes.
4. **Targeted edits**: Keep your file edits as small and surgical as possible.
